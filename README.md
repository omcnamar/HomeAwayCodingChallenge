# HomeAwayCodingChallenge
This application uses Seat Geek API to show user Events based on their search 
-Technologies used:
  -	Dagger
  -	Retrofit 
  -	Gson
  -	Okhttp3
  -	Butterknife
  -	recyclerview
  -	MVP Architecture
